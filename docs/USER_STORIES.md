## Epic 1: 用户注册与登录

### US-1.1 邮箱注册
**作为** 新用户
**我想要** 使用邮箱注册账号
**以便于** 创建我的个人账户并开始使用应用

**验收标准:**
- [ ] 用户可以输入邮箱地址
- [ ] 系统发送验证码到用户邮箱
- [ ] 用户输入验证码完成验证
- [ ] 用户设置密码（至少8位，包含字母和数字）
- [ ] 注册成功后自动登录并进入新手引导
- [ ] 邮箱格式验证和重复注册检查

**优先级**: P0 | **故事点**: 5

---

### US-1.2 手机号注册
**作为** 新用户
**我想要** 使用手机号注册账号
**以便于** 更方便地创建账户

**验收标准:**
- [ ] 用户可以输入手机号（支持国际区号）
- [ ] 系统发送短信验证码
- [ ] 验证码60秒内有效，可重新发送
- [ ] 验证成功后设置密码
- [ ] 手机号格式验证

**优先级**: P1 | **故事点**: 5

---

### US-1.3 账号登录
**作为** 已注册用户
**我想要** 使用账号密码登录
**以便于** 访问我的账户和聊天记录

**验收标准:**
- [ ] 用户输入邮箱/手机号和密码
- [ ] 密码错误提示（不暴露具体原因）
- [ ] 连续5次错误锁定15分钟
- [ ] 登录成功跳转到聊天列表
- [ ] 支持"记住登录状态"

**优先级**: P0 | **故事点**: 3

---

### US-1.4 OAuth第三方登录
**作为** 用户
**我想要** 使用第三方账号快速登录
**以便于** 无需记忆额外的密码

**验收标准:**
- [ ] 支持Manus OAuth登录
- [ ] 首次登录自动创建账户
- [ ] OAuth账号可绑定手机/邮箱

**优先级**: P1 | **故事点**: 5

---

### US-1.5 新手引导
**作为** 首次登录的用户
**我想要** 了解应用的基本功能
**以便于** 快速上手使用

**验收标准:**
- [ ] 展示应用核心功能介绍（3-5步）
- [ ] 引导用户选择感兴趣的角色类型
- [ ] 推荐一个系统角色开始第一次对话
- [ ] 完成引导奖励50积分
- [ ] 可跳过引导

**优先级**: P1 | **故事点**: 3

---

### US-1.6 忘记密码
**作为** 忘记密码的用户
**我想要** 重置我的密码
**以便于** 重新访问我的账户

**验收标准:**
- [ ] 通过邮箱/手机验证身份
- [ ] 设置新密码
